# Alberta Used Car Market Scraper

A robust Python-based data tool designed to scrape and clean private-owner car listings across Alberta from Kijiji.

## Key Features
- Province-Wide Coverage: Automated scraping across 11 major Alberta regions including Calgary, Edmonton, and Red Deer.
- Advanced Data Cleaning: Leverages Pandas and Regex to filter out non-vehicle listings such as parts, tires, and wanted ads.
- Market Monitoring: Tracks vehicle availability (Active vs. Sold) by cross-referencing daily crawls with a master database.
- Detailed Specification Extraction: Automatically captures Kilometres, Transmission, Drivetrain, and other core specs from individual listing pages.

## Tech Stack
- Language: Python
- Libraries: Requests, BeautifulSoup4, Pandas, Re (Regex)
- Data Management: Automated CSV-based master database
